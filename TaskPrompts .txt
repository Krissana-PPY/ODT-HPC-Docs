เขียน script:
ฉันจะ train PyTorch model โดยใช้ dataset ขนาด 50 GB script ใช้เวลาประมาณ 10-12 ชั่วโมงในการทดสอบบนเครื่องตัวเอง ช่วยเขียน job script ให้หน่อย

วิเคราะห์ทรัพยากร:
นี่คือ Python script ของฉัน [วาง code] ฉันควรขอ CPU และ RAM เท่าไหร่ และควรใช้ partition ไหน

แก้ปัญหา:
งานของฉัน FAILED นี่คือ error log [วาง log] เกิดจากอะไร

ปรับปรุง script:
sacct บอกว่างานของฉันใช้ RAM จริงแค่ 8 GB จากที่ขอไว้ 64 GB และใช้เวลา 45 นาที ควรปรับ script ยังไง [วาง script]

ถาม best practice:
ฉันมี array job ที่ต้องรัน 100 ครั้ง แต่ละครั้งใช้ไฟล์ input ต่างกัน ควรออกแบบ script ยังไงดี
